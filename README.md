# Employee Payslip System

A Full Stack Employee Payslip Generation System built using Django (Backend API) and React (Frontend).

# Tech Stack
- Django
- Django REST Framework
- React
- Bootstrap
- SQLite

# Features
- Add Employee
- View & Delete Employee
- Generate Payslip
- Leave-based Salary Calculation
- Print Payslip

# Salary Logic
Gross Salary = Basic + HRA + DA + Other Allowances  
Net Salary = Gross - (ESI + PF)  
Final Salary = Net Salary - Leave Deduction
